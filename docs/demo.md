# RemNote CLI Demo Has Moved

The `remnote-cli` command is now provided by the `remnote-mcp-server` package.

See the maintained demo:

https://github.com/robert7/remnote-mcp-server/blob/main/docs/demo.md

Migration:

```bash
npm uninstall -g remnote-cli
npm install -g remnote-mcp-server
```

The `remnote-mcp-server` package provides both:

```bash
remnote-mcp-server
remnote-cli
```
